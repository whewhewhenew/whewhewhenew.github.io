ARMOR

ARMOR 全称为 AI Robustness, Model security, and Offensive-Defensive of RobotOS。团队名称概括了我们的核心研究目标：围绕人工智能鲁棒性、模型安全以及机器人系统攻防，构建兼具理论深度与实践价值的研究体系。

Team Vision

ARMOR 致力于面向真实复杂场景中的智能系统安全问题，研究人工智能模型本身的鲁棒性与安全性，同时探索如何利用人工智能能力解决传统网络与软件安全问题。

Team Goal

围绕“模型可信、系统可防、攻防可验证”展开研究，推动 AI 安全与安全智能化的深度融合，服务于高可靠智能系统与机器人系统的实际部署。

Research Tracks

团队目前的研究方向主要可分为两类：AI security 与 AI for security。

AI Security

AI security 关注神经网络本身的脆弱性以及人工智能模型本身存在的安全问题，重点研究模型鲁棒性、安全测试、缺陷修复、对抗样本检测与防御，以及大模型与深度学习系统的安全机制。

AI for Security

AI for security 关注利用人工智能技术解决传统安全问题，例如漏洞挖掘、软件脆弱性检测、自动化安全分析、攻防推理与风险识别，探索多智能体与大模型在安全领域的应用潜力。

AI Robustness

研究模型鲁棒性评估、对抗攻击与防御、可靠测试与修复增强。

Model Security

关注深度学习模型与大模型的安全风险、缺陷定位与防护机制。

RobotOS Security

聚焦机器人操作系统中的通信、调度、感知与控制链路的攻防问题。
